##【基本要求】 利用单向链表实现学生信息的存储，每个结点包括：学号，姓名，语文成绩、数学成绩、英语成绩。

####【问题描述】 主要功能： 
####1、 学生信息录入； 
####2、 学生信息浏览； 
####3、 信息查询（a 按学号查  b 按最高分查（显示每门课的最高分的同学信息）） 
####4、 信息排序（a 按总分排序（从高到低）b 按姓名排序  ） 
####5、 信息删除（给定学号，删除该学生信息） 
####6、 信息修改（给定学号，修改该学生信息）
